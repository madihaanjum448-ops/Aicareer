# 🚀 CareerPilot AI

An AI-powered Career Guidance Platform that helps students analyze their resumes, identify skill gaps, and generate personalized learning roadmaps to achieve their dream careers.

---

## 📌 Overview

CareerPilot AI is designed to guide students who are unsure about their career path by using Artificial Intelligence to analyze their current skills and recommend the best learning journey.

Instead of manually searching for courses and required skills, users simply upload their resume, choose a target career, and receive AI-powered recommendations.

---

## ✨ Features

- 🔐 User Authentication (Login & Signup)
- 📄 Resume Upload (PDF)
- 🤖 AI Resume Analysis
- 💡 Automatic Skill Extraction
- 📊 Skill Gap Analysis
- 🎯 Career Goal Selection
- 🗺 Personalized AI Learning Roadmap
- 📈 Resume Score
- 👤 User Profile Dashboard
- 📱 Responsive Modern UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/madihaanjum448-ops/Aicareer.git
```

Move into the project

```bash
cd Aicareer
```

Install dependencies

Frontend

```bash
cd frontend
npm install
npm run dev
```

Backend

```bash
cd backend
npm install
npm start
```

AI Service

```bash
cd ai-service
pip install -r requirements.txt
uvicorn main:app --reload
```

---

## 🎯 Workflow

1. User logs in.
2. Uploads Resume (PDF).
3. AI extracts technical skills.
4. User selects a target career.
5. AI compares current skills with industry requirements.
6. Skill gaps are identified.
7. Personalized learning roadmap is generated.
8. Resume score is displayed.
